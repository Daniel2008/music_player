# 项目简介

## 🎵 Music Player

Music Player 是一个功能强大的跨平台桌面音乐播放器，基于 Flutter 框架开发，支持 Windows、macOS 和 Linux 系统。

## 项目背景

本项目旨在创建一个现代化、功能丰富的音乐播放器，结合本地音乐播放和在线音乐服务，为用户提供一站式的音乐体验。

## 核心功能

### 🎧 播放功能
- **本地音乐播放**：支持 MP3、FLAC、WAV、M4A、OGG 等常见音频格式
- **在线音乐**：集成多个音乐源（网易云、QQ 音乐等）
- **播放控制**：播放/暂停、上一曲/下一曲、进度跳转
- **音量控制**：独立音量控制，支持实时调节
- **播放队列**：完整的播放列表管理

### 📝 歌词系统
- **本地歌词**：自动识别同名 .lrc 文件
- **在线歌词**：智能搜索和匹配在线歌词
- **同步显示**：实时歌词滚动和高亮
- **翻译支持**：部分歌曲支持双语歌词

### 🎨 界面与主题
- **Material Design 3**：遵循最新的 Material Design 设计规范
- **深色/浅色模式**：支持主题切换
- **自定义皮肤**：通过 JSON 配置自定义主题
- **响应式设计**：适配不同屏幕尺寸

### 📊 音频可视化
- **频谱显示**：实时 FFT 频谱分析
- **波形显示**：音频波形可视化
- **全屏模式**：支持全屏可视化效果

### ⌨️ 快捷键支持
- **全局快捷键**：系统级媒体控制
- **Ctrl+Alt+P**：播放/暂停
- **Alt+左/右方向键**：快进/快退 5 秒

### ❤️ 收藏与历史
- **收藏管理**：收藏喜欢的歌曲
- **播放历史**：自动记录播放历史
- **快速访问**：快速访问收藏和历史

### 📥 下载功能
- **在线下载**：下载在线音乐到本地
- **音质选择**：支持多种音质选项（128k、320k、无损、Hi-Res）
- **下载管理**：查看下载进度和状态

## 技术特点

### 跨平台
- 基于 Flutter 框架，一套代码支持三大桌面平台
- 平台特定功能通过条件编译实现
- 原生性能和流畅体验

### 高性能
- 使用 SoLoud 音频引擎，低延迟高质量播放
- 异步加载和缓存机制
- 优化的 UI 渲染性能

### 可扩展
- 模块化架构设计
- Provider 状态管理，易于扩展
- 插件化的音乐源支持

## 项目结构

```
music_player/
├── lib/
│   ├── audio/           # 音频相关（均衡器等）
│   ├── models/          # 数据模型
│   ├── platform/        # 平台相关（快捷键等）
│   ├── providers/       # 状态管理
│   ├── services/        # 服务层（API 等）
│   ├── ui/              # 用户界面
│   │   ├── pages/       # 页面
│   │   └── widgets/     # 组件
│   ├── utils/           # 工具类
│   ├── app.dart         # 应用入口
│   └── main.dart        # 主函数
├── assets/              # 资源文件
│   ├── lyrics/          # 示例歌词
│   └── skins/           # 皮肤配置
├── docs/                # 文档
├── test/                # 测试
└── wiki/                # Wiki 文档
```
